flask,tornado,fastapi 的比较

大数据flask,tornado,fastapi三个都是web框架，前两者都十分流行，后者是一个新框架。

  • flask是一个轻应用框架，简单好用易扩展。

  • tornado 是一个非阻塞的异步框架，性能高，并发高。

  • fastapi是一个入门简单，性能高的框架，据说可比node 和go，仅支持 python3.6及以上
